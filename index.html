Game Now Crowdfunding site

"A Crowdfunding website for hopeful Game Developers - what more could you want? I first created the API endpoints using Django Rest Framework, then hooked it up to a React front-end. Features include login and signup, the ability to create a new Project, and user-guided features such as only being able to view a Profile or Pledge when signed in. Keep track with a running-total at the top of each Project!"

Scholarship Pro

"As our Group Project we produced a Scholarship website, with the target audience being "The Ones left Behind", being those with vision impairment, those in a racial or gender minority, or are even ESOL. Features of the website includes filtering Scholarships by category, Users being able to favourite their own Scholarships, and an optional Quiz so that a Scholarship Pro specialist can be in touch to connect them to a Scholarship opportunity."

Django News Site
